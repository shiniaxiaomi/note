# 使用说明

## PC端

快捷键

1. 搜索博客和标题关键字 `Alt+I`
2. 退出搜索 `ESC`
3. 搜索内容关键字 `Ctrl+F`

> 在PC端使用的频率较高,因此在使用体验上会高于移动端

## 移动端

> 用于在移动端上的浏览和查看,但是没有快捷键的支持,但是也提供了搜索功能,体验和外观比不上PC端
