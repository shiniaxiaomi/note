# 模块

## layer弹出层模块

快速使用:

- 在html中引入以下链接

  ```html
  <script src="https://cdn.bootcss.com/jquery/3.4.1/jquery.min.js"></script>
  <script src="https://cdn.bootcss.com/layer/2.3/layer.js"></script>
  ```

- 就可以直接使用了

  ```js
  layer.msg("正在打开Typora中...", {
      time: 1000 //1s后自动关闭
  });
  layer.msg("正在打开");
  ```
